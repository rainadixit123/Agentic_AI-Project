Multi-Agent Financial & Web Search Assistant

A Python-based AI assistant powered by Groq LLaMA 3.3, integrating DuckDuckGo web search and Yahoo Finance tools using the phi framework.
It supports two standalone agents and a combined multi-agent system capable of fetching real-time stock data, fundamentals, analyst recommendations, and latest news, along with web search capabilities.

A built-in Playground UI is also included for interactive testing.
